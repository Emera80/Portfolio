Bienvenue sur le dépôt de mon portfolio personnel. Ce projet est une vitrine interactive de mes compétences en développement Fullstack, de mes réalisations et de mon parcours académique. Conçu avec une approche "Mobile-First" et des animations fluides, il reflète ma passion pour le code propre et les expériences utilisateur exceptionnelles.

## 🚀 Fonctionnalités Clés

### 🎨 Design & Expérience Utilisateur
- **Interface Immersive** : Design moderne avec des effets de parallaxe, de flou dynamique au scroll et de "Glassmorphism".
- **Animations Fluides** : Utilisation intensive de **Framer Motion** pour des transitions douces, des apparitions en cascade et des carrousels interactifs.
- **Navigation Intuitive** : Scroll fluide (Smooth Scroll) entre les sections via une barre de navigation réactive.

### 💻 Sections Principales
- **Hero Section** : Une introduction visuelle forte avec un effet de texte superposé (3D-like).
- **About Me** : Présentation de mon parcours en tant qu'étudiant en informatique et développeur.
- **Skill Showcase** : Grille de compétences dynamique filtrable par catégorie (Frontend, Backend, Database, Tools).
- **Project Gallery** : Carrousel interactif présentant mes projets phares avec détails techniques et liens (GitHub/Live).
- **Awards & Achievements** : Timeline verticale animée retraçant mes succès (Hackathons, Certifications).
- **Contact Form** : Formulaire fonctionnel intégré avec **EmailJS** pour une communication directe sans backend dédié.

## 🛠️ Stack Technique

### Frontend
- **Framework** : React 19 (Vite)
- **Styling** : Tailwind CSS 4
- **Animations** : Framer Motion
- **Icônes** : React Icons, Lucide React
- **Gestion des Emails** : @emailjs/browser

### Outils & Déploiement
- **Versionnage** : Git & GitHub
- **Hébergement** : Vercel
- **Environnement** : WebStorm / VS Code

## 📦 Installation et Configuration

### Prérequis
- Node.js 18+
- Un compte [EmailJS](https://www.emailjs.com/) (pour le formulaire de contact)

### 📥 Clonage et Installation
```bash
git clone https://github.com/Emera80/portfolio.git
cd portfolio
npm install
```

### ⚙️ Configuration des Variables d'Environnement
Créez un fichier `.env` à la racine du projet et ajoutez vos identifiants EmailJS :
```env
VITE_SERVICE_ID='votre_service_id'
VITE_TEMPLATE_ID='votre_template_id'
VITE_PUBLIC_KEY='votre_public_key'
```

### 🚀 Lancement
```bash
npm run dev
```

## 📐 Architecture du Projet

Le projet suit une structure modulaire pour faciliter la maintenance :
- `src/components/` : Composants réutilisables pour chaque section du portfolio.
- `src/assets/` : Ressources statiques (images, icônes).
- `src/pages/` : Pages principales de l'application (SPA).
- `src/index.css` : Configuration globale de Tailwind et styles personnalisés.
